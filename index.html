About

What is Open Niji

Open Niji is a project which is a fusion between Niji journey and stable diffusion. Niji journey is an anime ai image generation with two faults. 1. It is not open source 2. It has several errors generating faces and hands. As such Open Niji was made. Open Niji is a stable diffusion model trained on Niji Journey images in order to create the ultimate image model.

Progress on v3

V3 is currently being trained from scratch on a stable diffusion model with over 100k, high quality Niji journey images being used in its development. Instead of using a base anime model, v3 is from scratch in order to create a model with exceptional quality. However, in order to create the best possible model, we need YOUR help, so please give us your best Niji journey images or personal artwork for training.
